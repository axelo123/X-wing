	====================================
	CE QUI EST DISPONIBLE POUR UN JOUEUR (voir les fichiers collection_*.txt)
	====================================

Utilisateurs :
* ceux qui possèdent le jeu :
	-> peut gérer les vaisseaux, les cartes pilote/vaisseau et les cartes d'amélioration
	   (la suppression et la modification peuvent poser des problèmes !!!)
	-> peut gérer sa collection :
		un certain nombre de figurines
		un certain nombre de cartes pilote/vaisseau
		un certain nombre de cartes d'amélioration
	-> peut créer un ou plusieurs escadrons (une liste d'armée) pour x points
* ceux qui ne possèdent pas le jeu :
	-> peut créer un ou plusieurs escadrons (une liste d'armée) pour x points
	
	========================================
	CE QUI EST ATTENDU COMME FONCTIONNALITES
	========================================
	
Gestion des utilisateurs :
- deux types d'utilisateurs :
	- collectionneur (joueur possédant le jeu)
	- invité (joueur ne possédant pas le jeu)
- description :
	- nom, prénom, pseudonyme
	- login/mot de passe

Pour les collectionneurs, gestion :
	- figurines (vaisseaux)
	- cartes pilote/vaisseau
	- cartes d'amélioration

Pour les collectionneurs, gestion de leur collection :
	- nombre de figurines/vaisseaux de chaque type
	- nombre de cartes pilote/vaisseau de chaque type
	- nombre de cartes d'amélioration de chaque type
	
Pour tout utilisateur, gestion de ses escadrons :
	- un nom et/ou un descriptif
	- le choix de la collection
	- le camp concerné (attention aux Racailles/scélérats qui peuvent compléter les forces de l'Empire ou des Rebelles)
	- quantité de points (maximales) = coût total
	- composition de l'escadron en respectant :
		- quantité de figurines/vaisseaux disponibles dans la collection
		- quantité de cartes pilote/vaisseau disponibles dans la collection
		- quantité de cartes d'émalioration disponibles dans la collection
		- toutes les règles de combinaisons

Fonctionnalité finale attendue :
	- générer à partir d'un escadron,
	  un document imprimable et/ou facilement consultable (par exemple sur smartphone)
