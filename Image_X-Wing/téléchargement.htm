Outil créé par oopinou pour l'Atelier du Faucon. Inspiré par le travail de Jason Fuller et basé sur les templates HD de Tom Howard.
Ce site n'est pas avalisé par Lucasfilm Ltd., Fantasy Flight Games ou EdgeEnt et a été créé dans le but de divertir. Star Wars, le logo Star Wars, ainsi que tous les visuels sont des marques commerciales enregistrées et/ou copyrights de leurs propriétaires respectifs.
Téléchargez votre image avec un clic droit, puis "enregistrer l'image sous ...",
si ça ne marche pas, essayez de recharger la page.




Pour voir comment imprimer vos images, cliquez ici.

Pour imprimer vos images, vous pouvez suivre les instructions suivantes :



- Téléchargez Xnview pour votre système d'exploitation et installer le.
- Ensuite, cherchez l'emplacement de l'image et ouvrez la avec un double clic :


- Une fois l'image ouverte, choisissez "Fichier/imprimer ..." :


- Choisissez l'option de mise en page "Unique - Personnalisé" :


- Choisissez la valeur correspondante* pour régler la largeur ET la hauteur d'impression :
    - Pour les cartes pilote : 8.85
    - Pour les petits socles : 4.86
    - Pour les gros socles : 9.18
    - Pour les roues de manoeuvre : 4.37
Laissez "Garder les proportions" cochée puis cliquez sur "Imprimer" ** :



* si vous trouvez des valeurs différentes intéréssantes, merci de me contacter
** pour les cartes, un gabarit d'impression est disponible pour faire des tests sans gacher d'encre, aller voir ici.
